3W 文化 · 寿险经营的底层活动量操作系统

代表人物：陈玉婷（保德信人寿首席寿险顾问 · 2020–2021 MDRT 大中华区主席 · IDA 终身会员）。「3W」= 每周 3 张新保单（3 policies a Week），她自 1992 年起持续 1,235 周以上不间断，创「世界华人 3W 历史纪录」。

① 3W 的内核

用「对」的心态做「对」的事情；回到保险最初——以「爱」为本的保障规划（爱、备爱、实践爱），实现「财务安全 × 心灵和平」。不靠爆发，靠稳定、可累积的周节奏：每周固定 3 张新保单，雷打不动。

② 配套方法论：7S + KASH

7S：Search 寻找 / Service 服务 / Sell 销售 / Study 学习 / Sit Plan 计划 / Structure Activity 固定工作时刻表 / Sales Manager 管理职。
KASH：知识 Knowledge · 态度 Attitude · 技巧 Skill · 习惯 Habit。七力并进、长期经营。

→ 团队长怎么用（专治留存难）

把「3W」设为新人的最低活动量纪律：用稳定、可预期的周节奏替代「暴涨暴跌」式业绩，让新人每月都有确定性进展、看见自己在成长——这与 Granum「10-3-1」活动量系统同源，专治留存难。陈玉婷座右铭：「知识与经验的分享是给人们最好的礼物」，把她的 3W 节奏 + 7S 拆解进新人 90 天陪跑，留存自然提升。

📊每日行动计划模板（Granum 活动量版）▶

给每位组员一张表，主管每天只看这一张，过程可预测、结果可控制。结合陈玉婷 7S（固定工作时刻表 / 计划 / 管理），把「自律」变成「系统」。

  • 10-3-1 规律：每 10 个合格准客户 → 3 个面谈 → 1 个成交（Granum 数据证明此比例恒定）。
  • 台阶理论（吴征宇）：把「10 米高墙」拆成 50 个 20cm 台阶，让新人每次只盯下一步，目标不再吓人；最低节奏锚定陈玉婷「3W＝每周 3 张」。

🤝增员面谈框架（Granum 六步转介绍法改编）▶

融入宋明华「心法论」（想要什么人便找到什么人）、朱美音「三高 + 善良是风水」、龙子明「做大做强做持久」与李佳蓉「用任务成就人才」——把「找人」升级为「筛选与孵化领导者」。

  1. 取得认同：回顾你为对方（或他认识的人）做过的事，确认他对保险价值的真实感受。
  2. 引入人选：自然提到一个你认识、可能适合的人，「能多聊聊他吗？」
  3. 收集画像：年龄、职业、家庭、收入规划、人脉特点——判断匹配度。
  4. 追问还有谁：「想到他时，您还想到别的合适的人吗？」链式扩列。
  5. 锁定首选：把最熟、最匹配的人放名单首位，约定引荐方式。
  6. 重复扩展：对每位都问「他熟的人里，谁最可能也想了解？」形成推荐人网络。
💡1 对 1 辅导话术（四种高频场景）▶

① 心态崩了 / 想放弃
先共情不评判：「这行前两年最难，我 21 年里也想过无数次放弃。」再给 Mandino 视角：「我会坚持，直到成功——坚持不是硬扛，是把'每天见一个人'这种小事做成习惯。」用 Stone 的「Do it now!」把恐惧交给行动；也可借桑杰「4 原则」——不把工作当工作、只见喜欢的人、给最好建议、不执着结果，让新人把销售变轻松。

② 连续被拒绝
Feldman 金句：「客户说'不'，意思是'现在不行'，不是'永远不'。跟进、跟进、再跟进。」带他复盘：是哪句没接住？下次怎么改？

③ 业绩下滑
用 Granum 逻辑：「业绩是活动量的滞后指标。我们先不看结果，看过程——这周见了几个人？」用吴征宇「台阶理论」把大目标拆小，先保陈玉婷「3W＝每周 3 张」的最低节奏，焦虑自然落地。

④ 对行业没信心
梅第式信念：「保险是上帝赐给人类最美丽的好礼物。我们不是在卖合同，是在帮一个家庭避免'无知的代价'。」用真实赔案故事重建意义感。

⑤ 组员没动力 / 躺平
陈嘉虎「询导」：用提问代替命令——「你最想达成什么？团队能帮你什么？」引导他自我驱动；李佳蓉「用任务成就人才」：把合适的人放进经委会 / 功能组，让他在真实任务中长本事、被看见；龙子明「Feel good do better」：多表彰、多带外出参会、建荣誉墙，让人「感觉良好」自然更起劲，用 IDA「件数文化」给每人一个小目标。

情景话术库 · 开口就能用

每一条话术都已嵌入本智能体的大师与名家方法论：Ziglar 四问隔离 / Feldman「不=现在不行」/ Granum 六步 / Stone「Do it now」/ 梅第意义感 / 甘道夫长期服务，以及华人名家——陈嘉虎询导、李佳蓉经委会、龙子明件数文化 + Feel good、宋明华心法论、朱美音三高、吴征宇爱与责任、桑杰快乐销售。点开任一手风琴即可照读。

🛡️客户说「我考虑考虑」▶

Ziglar 四问隔离法：把模糊的拖延变成具体的议题——

  • 「为了理清我自己的思路，您想考虑的是哪一点？是怀疑我们公司？」
  • 「还是怀疑我这个人？」
  • 「那一定是产品本身了？」
  • 隔离出真问题后，再进入：「您喜欢吗？想要吗？买得起吗？想什么时候开始享受保障？」

Feldman 故事法：用一个有寓意的小故事，扩大「拥有的快乐」与「没有保险的痛苦」，让客户自己算账，而不是你催他。

🔁转介绍开口▶

Gandolfo 金句：「我需要你的帮助。」——真诚向客户求助，没有人会拒绝。比「您有朋友需要保险吗」自然百倍。

Granum 六步：先取得服务认同 → 提及具体人选 → 收集资料 → 追问还有谁 → 锁定首位 → 重复扩展。关键：在服务交付后自然索取，而非成交时才想起来。

🌱新人破冰 / 被拒绝时▶

Stone「Do it now!」：离开上一扇门，立刻对自己喊一句「现在就做！」走进下一扇。用行动压过恐惧（哈佛 James 原理）。

Feldman「不=现在不行」：把每次拒绝记下来，标注「回访日」，3 个月后、6 个月后、17 个月后都可能是成交点。

❤️传递行业意义（对抗职业倦怠）▶

梅第：「我年轻时每月做 2 份保单，现在每月参加 2 个葬礼——用当初苦口婆心让他们买的保单，给受益人送去几百万。在所有服务过的家族里，我最受尊敬。」

甘道夫：「我有义务为他们服务一辈子。」小客户 19 岁买 5000 保额，后来继承家业买了 720 万——客户是在和你一起成长的。

🔥怎么让组员有动力（团队长对内）▶

陈嘉虎「询导」：别说教，用提问引导——「你最想达成什么？团队能帮你什么？」让他自己说出动力。

李佳蓉「用任务成就人才」：把合适的人放进经委会 / 功能组，让他在真实任务中长本事、被看见，比空洞打鸡血更持久。

龙子明「Feel good do better」：多表彰、多带伙伴外出参会、建荣誉墙——让人「感觉良好」，做事自然更起劲；用 IDA「件数文化」给每人一个自己的小目标。

🌟招募面谈怎么说（吸引优才）▶

宋明华「心法论」：你想要什么样的人，就会找到什么样的人——先校准自己要的人，再谈招募，人才质量自不同。

朱美音「善良是最好的风水」+ 三高：用「和优秀的人一起做有意义的事」吸引对的人，而非只谈佣金。

吴征宇「传递爱与责任」：把保险讲成「用金融契约把爱和责任传下去」，让优才看见职业尊严与长期价值。

💎客户经营做深（终身服务）▶

梅第 / 甘道夫：把「卖一单」升级为「服务客户一辈子」——小客户也会长大，和他一起成长，续期与转介绍自然来。

苏婉薇「一站式理财策划」：用长期、专业、一站式服务锁定客户终身价值，降低脱落。

吴征宇「100% 转介绍」：售后远重于售前，诚信专业带来链式转介绍——也是团队活动量的源头活水。
