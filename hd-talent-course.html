COURSE OVERVIEW

天赋才华课程 · 总览

人类图中，闸门是能量入口，通道是连接两个中心、让天赋流动起来的"电路"。
36 条通道分属四大回路，各自回答一个关于天赋的根本问题：
我的能量，是来表达自己、分享群体，还是支持所爱之人？

🎯 天赋筛选器 输入你人类图里"有颜色"的闸门编号（如 1 8 10 20 34 57），高亮属于你的天赋通道
✓ 完整通道（双闸门都在）　◐ 悬摆闸门（仅一个，等待被遇见）
整

整合回路

INTEGRATION

独立于三大回路之外，服务于"自我定位"与生存本能。回答：我是谁、我如何在此刻真实地活着。

个

个体回路

INDIVIDUAL

突变、独特、不随波逐流的频率。天赋策略：爱自己、成为自己、活出自己，就能启发他人。

集

集体回路

COLLECTIVE

面向更大群体的分享与逻辑。天赋策略：藉由经验、分析与整理，把实用方法无条件分享给他人。

家

家族回路

TRIBAL

面向家庭与亲密社群的支持与连结。天赋策略：在让你有归属感的小团体里，给予实质的支持。

ENERGY CENTRES · 标准术语

九大能量中心

人类图盘由 9 个能量中心构成（源自卡巴拉生命树与脉轮系统）。
每个中心要么"有定义"（稳定能量），要么"开放"（吸收外界能量）。
以下为课程统一采用的标准中文术语与定义：

有定义：能量稳定、持续、可被信任 开放：吸收外界能量、灵活、易受影响
🧠

头脑中心

HEAD

思维与灵感的发源地，产生问题与困惑，驱动人类探索意义。

有定义：问题稳定，专注探索特定领域。　开放：易被外界问题淹没、过度思考。
包含闸门61 · 63 · 64

有定义：有自己固定的思考方式与信念。　开放：易吸收他人观点，思考随外界波动。
包含闸门4 · 11 · 17 · 24 · 43 · 47

表达与显化核心，所有能量最终流向喉咙以表达或行动。

有定义：表达稳定、言出必行。　开放：易多话、为说而说，或被他人"借嘴"说话。
包含闸门8 · 12 · 16 · 20 · 23 · 31 · 33 · 35 · 45 · 56 · 62

COURSE DESIGN

课程设计 · 教学大纲

建议以"四大回路"为四大模块推进，每模块 1.5–2 小时，可单场 6 小时集中版，或分 4 周深度版。配套《天赋才华学员手册》与通道速查卡。

模块主题核心内容教学目标
模块一整合回路 · 出厂核心10-20 / 10-34 / 10-57 / 20-34 / 20-57 / 34-57：自我定位、活在当下、直觉与行动合一的生存本能。锚定"我是谁"，建立稳定的天赋输出底座。
模块二个体回路 · 独一无二的我1-8 / 2-14 / 3-60 / 12-22 / 23-43 / 24-61 / 25-51 / 28-38 / 39-55：创意、突变、天才洞察与灵性深度。接纳自己的"不一样"，把独特活成天赋。
模块三集体回路 · 分享与贡献4-63 / 5-15 / 7-31 / 9-52 / 11-56 / 13-33 / 16-48 / 17-62 / 18-58 / 29-46 / 30-41 / 35-36 / 42-53 / 47-64：逻辑、领导、说书、精通。找到自己在群体中的贡献方式，把经验变方法。
模块四家族回路 · 支持与连结6-59 / 19-49 / 21-45 / 26-44 / 27-50 / 32-54 / 37-40：亲密、需求、金钱、守护、归属。在关系与社群中，用天赋给予实质的支持。
模块五综合 · 我的天赋地图结合个人人类图，定位"已定义通道 / 悬摆闸门"，绘制个人天赋地图与 21 天天赋实验。从"知道天赋"走向"活出天赋"，形成可持续练习。
HOW TO USE

天赋识别 · 如何使用本课程

天赋不是"全有或全无"。在人类图里，通道有"已定义"与"悬摆闸门"之分，理解它们，才不会误判自己的才华。

✓

已定义通道

DEFINED CHANNEL

两端闸门都有颜色，能量稳定、持续、可被信任。这是你出厂即带的天赋，越活越熟，是身份中可靠的一部分。

📌 看图：两端闸门都被涂色（实心）的通道＝已定义，能量稳定、持续可被信任。

◐

悬摆闸门

HANGING GATE

只有一端有颜色，另一端在等一个"对的人"来补全。它让你被他人启发、向他人学习，并自然吸引互补的人进入生命。

📌 看图：仅一端闸门有颜色、另一端空白，即为悬摆闸门，等待被遇见。

⚡

格式通道

FORMAT CHANNELS

3-60 / 9-52 / 42-53 连接根部与荐骨，是"最强驱动力"。一旦定义，会深刻影响你整个生命的节奏与主题。

📌 看图：根部中心—荐骨中心之间加粗连线的 3 条（3-60 / 9-52 / 42-53）＝格式通道。

🧭

找自己的步骤

3 STEPS

① 拿到自己的人类图；② 在上方"天赋筛选器"输入有颜色的闸门；③ 顺着高亮的通道，读对应的天赋解读文章。
