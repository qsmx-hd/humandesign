INTRO

课程引言：这套方案解决什么

这份方案解决什么：把人类图作为"读人"的科学工具，嵌入机构既有的优增优育痛点解决方案。原体系把"术"做得很满——四大痛点、话术、打卡、打分、SOP 全是动作层；但全是标准化，没解决"对谁标准化"。人类图补这一刀：把"统一话术"升级成"统一框架下的千人千面"。

定位为面向保险中支机构的"工具 + 培训"产品包；应用深度取轻量版四要素——主管只需会看"类型 / 权威 / 定义 / 人生角色"四样，即可落地，无需懂人类图全部知识体系。

MODULE 01

底层逻辑：术 + 道

原体系补"术"：渠道拓新、脱敏邀约、异议标准、面试闭环——解决"说什么、做什么动作"（已落地为六维评估、11 大赛道、70-30 法则、面谈五步法、追踪表）。

人类图补"道"：在统一框架下按对方的能量类型 / 决策权威 / 定义 / 人生角色做个性化适配——解决"对谁说、怎么说才命中、该攻哪类市场"。

轻量四要素的分工：
类型 解决"这人适合干哪块、怎么带" → 对应人才画像 / 师徒匹配
权威 解决"怎么跟他说、何时促单" → 对应脱敏邀约 / 促成时机（与 70-30 法则协同）
定义 解决"他是单兵还是靠团队" → 对应师徒搭配 / 留存环境
人生角色 解决"关系经营方式 / 天生渠道" → 对应 缘故市场 vs 陌生市场（与 11 大赛道协同）
MODULE 02

优增侧（招募）

2.1 能量人才画像升级（叠加六维人才评估模型）

保留原职业身份画像（店长 / 宝妈 / 企业内勤），叠加能量类型适配度，避免把"反映者"硬拉进高压节奏（注定流失），把对的人放在对的角色。

能量类型占比 / 特征在优增优育中的角色
生产者 / 显示生产者~70%，持续行动力销售主力 · 直增 · 大量触达 · 有回应就干
投射者~20%，需被邀请 / 看见顾问 · 陪访 · 教练 · 深度一对一；不宜硬压 KPI
显示者~8%，发起开拓渠道开拓尖兵 · 启动新项目；需告知计划 + 给自主
反映者~1%，需稳环境特殊关怀 · 低频；一般不强推增员
人类图 × 六维 互补对照：六维评"标准分"，人类图补"能量成因"，双视角降低错配流失。（六维共 6 项，覆盖行业认知、学习能力、人脉资源、意愿度、抗压、沟通）
· 行业认知（基础项） ↔ 类型 / 权威：六维看"是否懂保险 / 金融语境"；人类图看"靠哪种方式建立认知"——显示生产者靠快速试错上手、投射者靠深度研究后再输出，错配的教学方式会拖慢其行业认知。
· 学习能力（核心项） ↔ 类型 / 定义：六维看"能否持续学新规、新产品"；人类图看"学习节奏"——生产者靠"做中学"、投射者靠被邀请参与深度研讨、四分人靠团队带教补位，单兵硬学易流失。
· 人脉资源（加分项） ↔ 人生角色含 4 爻：六维看"人脉有多少"，人类图看"天生靠不靠人脉"——含 4 爻者让人脉自然裂变，逼其陌生地推是浪费。
· 意愿度（一票否决项） ↔ 权威 / 定义：意愿度 ≤2 直接"暂不适合"；人类图解释"为什么推不动"——如情绪权威被逼当场决定会本能抗拒。
· 抗压能力（核心项） ↔ 类型 / 定义：六维看恢复力；人类图看"是否被错配消耗"——投射者被压硬指标、显示者不被告知计划，抗压会被结构性击穿。
· 沟通能力（核心项） ↔ 类型：投射者需被看见、显示者需自主、生产者要"有回应的行动"，沟通打法因类型而异。

2.2 按权威定制脱敏邀约

标准作业要求"全员统一脱敏、三不原则"（不提卖保单 / 不提入职 / 不提销售）。底线不动，但按对方的决策权威微调话术内核，命中其决策方式，见面率直接提升。

决策权威决策特征脱敏邀约话术要点（守三不原则）促成时机
情绪权威情绪需时间沉淀才清晰，当下易摇摆"先发你一份本地赛道报告，过两天清晰了咱再聊"严禁当场逼单，留白 1–2 天再跟
荐骨权威身体即时回应（嗯 / 啊）"你来试一次这个模式，看身体有没有能量"当场有回应即约下次
直觉权威（脾中心）当下直觉、瞬间警觉"你第一眼感觉这事儿靠谱不？"借当下直觉切入
意志力中心权威（自我权威）意志 / 承诺驱动，说到做到"愿不愿意给自己一个 3 个月承诺试试？"用挑战 / 承诺锁定
G中心权威（自我投射权威）需被"看见 / 邀请"才表达方向"我特别想请你来聊聊，因为你这特质很少见"多给其说话、确认方向的空间，邀请式推进
无内在权威（环境权威）看环境对不对，无内在指南针"来现场感受下我们团队氛围"多次接触、让其感受环境后再自然推进
月亮循环权威（反映者）需一整个月亮周期（约 28 天）才看清"先感受、慢慢看，不急着决定"低频、给足时间，约 28 天周期评估，不强推
注：人类图标准权威共 7 种，本表已覆盖全部。原"脾权威 / 自我权威 / 环境权威"在此对应"直觉权威（脾中心）/ 意志力中心权威（自我权威）/ 无内在权威（环境权威）"——名称标准化，含义不变；新增的 G中心权威（自我投射权威）、月亮循环权威（反映者）补全了标准全集，邀约时按对方实际权威选用即可。

2.3 人类图面试法（看图面试）

把标准化面试 SOP 升级为"看图面试"，让人类图成为"免费职业测评"的完美载体，且天然契合脱敏人设。它架在机构既有的"面谈五步法 + STAR 结构化面试 + 70/30 法则"之上，不是另起炉灶。前置动作：加微信后先发人类图小程序测评链接（出生时间即可生成图）——即"我帮你做免费职业测评"的落地载体，面试前你已拿到对方的四要素图。

2.3.1 面谈五步法实操

总框架：① 共鸣（30秒）→ ② 共情（2-3分钟）→ ③ 拆困（5-8分钟）→ ④ 重构（5-8分钟）→ ⑤ 邀约（2-3分钟）。核心原则：整个面谈外勤说 30%、候选人说 70%（70/30 法则）；信任 = 让对方放下戒备、进入对话状态。

步骤时长核心原则关键动作 / 话术模板人类图叠加（看图面试）
① 共鸣30秒 不谈保险，先谈对方行业和近况 三模板：行业共鸣型（"最近你们行业怎么样？听说XX政策 / 市场变化挺大的"——适用报行合一 / 教培双减 / 医药带量采购）；生活关怀型（"看你朋友圈最近在忙XX，生活状态挺好的啊"）；第三方引荐型（"XX跟我提到你很专业，想听听你的看法"） 用类型 / 权威开场建专业感（如"你是生产者、荐骨权威，行动力天生强，跟销售很搭"），不暴露招聘目的
② 共情2-3分钟 先听后说，让对方感到被理解 三问："你现在工作中最大的挑战是什么？" → "你觉得现在这个状态能持续多久？" → "如果有一个机会能改变现状，你会考虑吗？"。技巧：不急着给建议；用"我理解 / 确实"回应；找情绪触发点（收入下降？晋升无望？时间不自由？） 按权威调倾听节奏：情绪权威多留白、不逼答；荐骨权威让其"试"；直觉权威（脾中心）借当下直觉接话
③ 拆困5-8分钟 用提问引导对方自己发现问题（非直接给答案） SPIN 提问法：S 现状（"你收入结构什么样？底薪占比？"）→ P 痛点（"报行合一再收紧，收入怎么变？"）→ I 影响（"持续下去对家庭 / 孩子教育 / 退休规划有何影响？"）→ N 需求回报（"有方式在保持现状同时增一份不受政策影响的收入，愿了解吗？"）。另按赛道选 1-2 个拆解问（银行 / 企业主 / 宝妈 / 退休女性 / 青二代 / 通用） 结合人生角色定向挖：含 4 爻问"你身边谁可能需要"；含 5 爻问"你公域影响力怎么用"；偏 1/3 问"你最擅长的专业是什么"
④ 重构5-8分钟 不是推销，帮对方看到新路径 跨行业对比 6 维度：收入结构（底薪 + 绩效有天花板 vs 佣金 + 管理津贴上不封顶）/ 时间自由度 / 成长路径 / 风险特征（单一 vs 多元抗风险）/ 客户关系（一次性 vs 终身）/ 创业门槛（资金场地 vs 零成本启动）。话术示范（银行）："同样服务高净值客户，保险变现效率是银行 3-5 倍" 用图背书重构："所以你适合顾问角色 + 缘故渠道"——图谱比口头承诺可信，避免画饼
⑤ 邀约2-3分钟 自然过渡到"了解一下" 三种邀约：轻邀约（"就当多一个选择，不合适也没关系"）/ 价值邀约（"我朋友 / 团队做得很好，介绍你们认识"）/ 体验邀约（"下周有个沙龙 / 下午茶，像职业发展分享会"）。721 跟进法则：7天内 ≥3 次触达、促成 ≥1 次深入面谈（24h 感谢 + 邀请 / 3天分享文章 / 7天约二次面谈） 看图促成：用类型说明适配角色、人生角色指明主攻市场（见 2.4），当场双向筛选避免错配；按权威定促成时机
三不原则（守底线）：不夸大 / 不被动 / 必闭环；看图面试新增——用图说话，不画饼。异议处理万能公式：表面顾虑 → 底层痛点 → 对应赛道方案（如"我再想想"→ 收入 / 面子 / 家人 → 追问真实顾虑；"保险不好做"→ 刻板印象 → 行业数据重构；"我有自己的事业"→ 安全感 → 加安全垫 / 对冲）。

2.3.2 STAR 结构化面试

用过去的行为预测未来表现——通过问候选人真实经历（Situation 情境 / Task 任务 / Action 行动 / Result 结果），判断其是否具备保险行业所需素质。人类图"面谈前已拿到"，可在每个维度定向追问。

维度目的标准提问示例人类图叠加（定向追问）
S 情境了解职业状况"你现在的工作 / 事业是什么情况？做了多久？"结合类型：显示者问"你发起过什么"；投射者问"你等过谁邀请"
T 任务评估核心成就"你在那项目具体负责什么？目标是什么？"按权威：荐骨权威问"你身体最想做的那件事"
A 行动评估行动力 / 执行力"面对困难你具体做了什么？怎么推动的？" 区分团队与个人："你个人具体做了什么？"按定义：一分人问"你单兵怎么打"；无定义问"你和谁搭才成"
R 结果判断结果导向 / 自驱"最后结果怎样？若重来一次你会怎么做？" 抗压探查："你遇到最大挫折是什么、怎么处理的？"结合人生角色：含 6 爻问"长期信任怎么建"；偏 1/3 问"专业沉淀出什么成果"
STAR 评分表（对接六维）：候选人 | S | T | A | R | 对应六维维度（行业认知 / 沟通 / 人脉 / 学习 / 抗压 / 意愿度）| 评分。抗压直接由 R 的"挫折处理"项评估；意愿度一票否决（≤2 否），抗压 ≤2 先补，六维 ≥3 且意愿 ≥4 直接送认证。人类图不替代六维，而是标注"他会在哪类任务上自燃"。

2.3.3 双工具收口（六维定"送不送" × 人类图定"放哪"）

面谈后独立填六维评估定"是否送认证"（意愿度 ≤2 否、抗压 ≤2 先补、六维 ≥3 且意愿 ≥4 直接送）；面谈前已拿到的四要素图定"放哪个角色 / 渠道"——两者各管一段，叠加成完整面试闭环。红线：不夸大 / 不被动 / 必闭环 / 用图说话不画饼。

2.4 人生角色（Profile）：缘故市场 vs 陌生市场 · 新增维度

行业第一大痛点，是"渠道枯竭、缘故透支、无法触达优质人才"。传统做法是让所有人硬啃缘故，导致资源透支、且不适合缘故的人极其痛苦。人类图人生角色（Profile）能直接判断：这个人天生靠"人脉连接"还是靠"破圈影响"——从而把对的人放到对的渠道，从根上解决"缘故透支"。

Profile 是什么：由两个爻线数字组成（个性 / 意识爻线 · 设计 / 潜意识爻线，各为 1–6），数字含义——1 研究者（基础建立者 · 知识积累）｜2 隐士（天才等待者 · 需被召唤）｜3 殉道者（实验学习者 · 试错韧性）｜4 机会主义者（关系织网者 · 人脉连接）｜5 异端（实用解决者 · 公众传播）｜6 角色模型（三阶领袖 · 长期信任）。人生角色（Profile）共 12 种（见 2.4.1）。

判断口诀：看 Profile 里含哪个"爻"——偏 1/3 → 实做型 → 陌生市场靠专业（研究 + 试错，靠专业内容破圈）；含 2 → 桥梁型 → 被引荐入圈（天生才能需被"看见 / 邀请"，安排引荐人带入，别让其独自破冰）；含 4 → 人脉型 → 缘故市场；含 5 → 传播型 → 陌生市场 / 公域；含 6 → 口碑型 → 长期信任。

2.4.1 人生角色 × 11 大赛道 × 市场类型 综合矩阵

把人类图"天生渠道偏好"与"11 大赛道"的职业身份画像交叉，得到可直接落地的分渠道增员地图。赛道匹配度评估表解决"他是哪类职业人群"，人生角色解决"他天生该走人脉还是破圈"——两者一起，渠道分配从盲打变双保险。

2.4.2 三类市场 SOP 工具包（转介绍 SOP / 内容-公域 SOP / 标杆打造 SOP）

增员 / 培育落地时，按对方人生角色推导出的市场类型，自动派发对应 SOP——缘故型（含 4 爻 / 含 2 爻桥梁型）发"转介绍 SOP"；陌生型（含 5 爻 / 1-3 实做型）发"内容 / 公域 SOP"；口碑型（含 6 爻）发"标杆打造 SOP"。三类 SOP 与"人生角色 → 市场"严格对应、互不串用，主管照单执行即可，无需自己摸索打法。

① 转介绍 SOP（缘故 / 人脉型专用 · 含 4 爻 · 含 2 爻桥梁型）

  1. 关系盘点：画出三层人脉圈——强连接（亲友 / 同事）/ 弱连接（老客户 / 同好）/ 潜在连接（朋友的朋友），标注"谁可能需职业转型 / 副业"。
  2. 价值前置：先给后求。用"免费人类图职业测评 / 本地赛道分析"做钩子，不卖货、不提入职（守三不原则），让人脉自然感受到价值。
  3. 触点设计：借节点邀约——生日、节日、行业变动、孩子升学等生活时刻，轻量触达而非硬推销。
  4. 脱敏转介绍话术：守三不原则（不提卖保单 / 不提入职 / 不提销售）。例："我最近在帮朋友用人类图做职业规划，免费帮你测一下？不涉及保险，就是看适不适合多一条路。"
  5. 熟人见证 / 案例背书：用身边真实故事破冰（"你认识的 XX 就是这么转型的"），比自夸可信。
  6. 裂变闭环：老客户转介绍激励 + 社群信任运营（宝妈群 / 退休姐妹团），让转介绍自循环——含 4 爻者最擅长此道，只需给舞台。

② 内容 / 公域 SOP（陌生 / 传播型专用 · 含 5 爻 · 1-3 实做型）

  1. 人设定位：选一个垂直赛道做深（如"宝妈理财避坑""理赔真相""职业转型实录"），专业 / 真实 / 垂直三选一主打，别贪全。
  2. 平台矩阵：按人群选阵地——视频号（私域 + 中年）/ 小红书（女性 / 宝妈）/ 抖音（泛流量）/ 知乎（专业纵深）。含 5 爻者天然会"被看见"，给素材即可。
  3. 内容选题：保险科普 / 避坑指南 / 真实理赔案例 / 人类图职业测评故事，每周稳定输出，用专业破圈而非人脉。
  4. 公域引流：评论区互动 + 私信钩子 + "免费测评领取"，把陌生流量引入私域。
  5. 私域承接：朋友圈人设运营 + 社群信任 + 一对一咨询，完成从"陌生"到"信任"的过渡。
  6. 转化节奏：不强攻，套用 70-30 法则——让对方说 70%、你只说 30%；按权威调追问节奏（情绪权威留白、荐骨权威让其"试"）。

③ 标杆打造 SOP（口碑 / 长期信任型专用 · 含 6 爻）

  1. 标杆定位：选其最擅长的高端客群（企业主传承 / 专业服务高端），树为团队标杆案例，让其"活成榜样"即是最好的增员名片。
  2. 长期信任运营：不急功近利，用"熟人见证 / 案例背书"打法深耕，时间换口碑；含 6 爻者成熟后影响力才达峰值，给足耐心。
  3. 内容沉淀：将其真实服务故事做成标杆素材，反哺团队其他成员的转介绍 / 公域 SOP，形成正向循环。
注：含 2 爻桥梁型虽归"被引荐入圈"，实际经营仍以缘故 / 转介绍为主（需引荐人带入圈后再走转介绍 SOP）；含 6 爻口碑型走"标杆打造 SOP"，节奏更慢、不急功近利，复用转介绍 SOP 的"熟人见证"打法。三类 SOP 已同步内置进独立的「人类图 × 保险职业适配系统」网页（见下方工具栏），主管输入四要素即自动派发对应一种。

3.1 类型化培育路径

培育端痛点是"留存率低"。留存率的钥匙在"满足类型的核心需求"——按类型给培育节奏，而非一刀切。

类型核心需求培育节奏建议
生产者 / 显示生产者有回应的行动 + 认可给明确任务、大量实践、及时正反馈；"等待回应"= 做有回应的事，不是不行动
投射者被看见、被邀请频繁给予反馈与认可、邀请其做顾问 / 陪访；缺认可极易流失
显示者自主空间告知计划、给发挥余地、不微观管控；发挥其开拓力
反映者稳环境、低压力低频接触、避免强推；一般不建议作为主增对象

3.2 心态脱敏新解

行业普遍以"拒绝 = 双向不匹配"做心态训练。人类图让其从"口号"变"可解释的科学"：被拒不是我不好，是我 / 对方的类型能量不匹配。把"卑微求人"转成"科学筛选"，内在真正接受。

3.3 师徒匹配

不同权威 / 类型的主管，带不同类型新人效果不同：

  • 投射者主管 → 擅长深度一对一辅导，适合带需要被"看见"的投射者新人。
  • 生产者主管 → 擅长带节奏、给任务，适合带生产者新人。
  • 避免同型短板叠加：如情绪权威主管带情绪权威新人，双方都"不过夜不决策"，易拖延——需机制补位。
  • 补充：按市场属性搭师徒——缘故型师傅带缘故型新人、陌生型（含 5 爻）师傅带陌生型新人，打法同频，带教效率最高（与 2.4 渠道分轨一致）。
